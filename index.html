FAQs:

1) Where to find Webhook URL?

Go to Server Settings; Find Integrations below Audit Log; Go to Webhook in Integrations and create one Webhook; Copy Webhook URL and paste it at WebhookURL Input Box


2) Is it safe for my server?

Ofcourse! It can only send messages; It cannot ban or kick any member; Make sure to keep WebhookURL private


3) Will Discord ban me if I use this?

No! Discord will not ban you; Discord has offcially given permission to send messages with Webhook
